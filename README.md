# Little Ummah – Crochet & Handmade Gifts
Little Ummah is a simple, elegant showcase website for handmade crochet items and soft toys. The project is still in progress, with new sections and product pages being added as the collection grows.

# Features
Auto‑scrolling gallery displaying featured crochet pieces

Clean, minimal layout focused on product visuals

Mobile‑friendly design for smooth browsing on all devices

Lightweight HTML/CSS/JS setup for fast loading and easy updates
